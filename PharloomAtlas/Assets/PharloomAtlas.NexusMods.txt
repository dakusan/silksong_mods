[*][b]Icons, Markers &amp; Pins[/b][list]
[*][b][img=middle]https://www.castledragmire.com/silksong/BenchSmall.png[/img] Icons[/b][list]
[*]All notable location data imported from [url=https://mapgenie.io/hollow-knight-silksong/maps/pharloom]MapGenie.io[/url] and displayed on the in-game map.
[*]Progress tracking — found icons automatically become invisible.
[*]Icon categories and category groups can be toggled through 3 states:
    Show all (found icons appear translucent/greyed) • Show incomplete • Hide all
[*]Detailed info with images for each icon on sidebar.
[*]Multiple icon sets supported — add your own by placing “[b]Icons-*.png[/b]” files in the plugin directory.

[*]Recenter the map on your character instantly.
[/list]

[*]Tunable panning speeds for both normal and marker modes, with unlocked borders for full freedom.
[*]Stops map position from jumping around when switching between marker modes.
[*][i]Note:[/i] The “wide” (general-area view) map is not supported.

All interfaces support 🎮️ controller, ⌨️ keyboard, and 🖱️ mouse for every feature.
[size=1](Search window does not support controller input.)[/size]
All keyboard bindings are customizable in the config.

[b]Map Mouse Controls:[/b][list]
[*]Scroll wheel — Zoom in/out
[*]Left-click drag — Pan the map
[*]Left-click icon — Show info on sidebar
[*]Hover over icon — Show title on sidebar
[*]Left-click marker or its label — Add/edit label (deleted markers retain labels until save reload). [size=1]The marker has to be in the exact correct position to restore the label.[/size]
[*]Right-click — Add/delete marker
[i]Note:[/i] Marker placement can be finicky near clustered markers — Team Cherry’s original marker auto-selection logic wasn’t built for mouse use.
[/list]
Via config you can:[list]
[*]Auto-enable mouse when sidebar or Saved Values windows are open
[*]Toggle mouse manually via the included [i]Plugin Developer Tools[/i] plugin (default: [b]F10[/b])
[/list]

To link icons and values:[list]
[*] Select an icon on the map (green highlight) and a value in the Saved Values window.
[*]Click [b]Save[/b] or [b]Save+Send[/b]:[list]
[*][b]Save[/b] → Local JSON
[*][b]Send[/b] → Uploads data to me for crowdsourcing
[/list][/list]
This will also improve icon alignment (MapGenie coordinates aren’t always precise).
If this goes well, I’ll automate for in-game JSON downloads — but there will [b]never[/b] be auto-updates or external calls without consent.

This is the only window that persists without the map open.
🎮️ Controller fully supported here too.
Some values are hard to identify — e.g., some secrets are hidden behind “*Remasker*” flags.
