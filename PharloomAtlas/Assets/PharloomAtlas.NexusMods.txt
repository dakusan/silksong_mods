[*]Everything is optional and configurable ([b]F1[/b]) — toggle features to suit your playstyle.
Enabled by default: [i]Sidebar &amp; map icons[/i] + [i]Highlight Hornet pin[/i]
[*]Full support for 🎮️ controllers, ⌨️ customizable keyboard keys, and 🖱️ mouse.[list]
[*]Zoom   [img=middle]https://www.castledragmire.com/silksong/CoinSuperSmall.png[/img]   Pan   [img=middle]https://www.castledragmire.com/silksong/CoinSuperSmall.png[/img]   Place+Label Markers   [img=middle]https://www.castledragmire.com/silksong/CoinSuperSmall.png[/img]   Select Icons   [img=middle]https://www.castledragmire.com/silksong/CoinSuperSmall.png[/img]   Sidebar   [img=middle]https://www.castledragmire.com/silksong/CoinSuperSmall.png[/img]   Save Values
[/list]
[*][b]Icons, Markers &amp; Pins[/b][list]
[*][b][img=middle]https://www.castledragmire.com/silksong/BenchSmall.png[/img] Icons[/b][list]
[*]All notable location data imported from [url=https://mapgenie.io/hollow-knight-silksong/maps/pharloom]MapGenie.io[/url] and displayed on the in-game map.
[*]Progress tracking — found icons automatically become invisible.
[*]Icon categories and category groups can be toggled through 3 states:[list]
[*][b]Show all[/b]: Displays everything; found icons translucent/greyed (config: [i]Found icon color [HSL][/i]).
[*][b]Show Normal[/b]: Hides found icons; hide unaccessible [size=1](config: [i]Hide unaccessible icons[/i])[/size] and undiscovered area [size=1](config: [i]Show undiscovered icons[/i])[/size] icons. [i][Both coming soon][/i][list]
[*]Full data sets incomplete; requirements/needs experimental—see [b]Saved Values Window[/b] below.
[*][i]“Hide unaccessible icons” off by default until databases complete.[/i]
[/list]
[*][b]Show None[/b]: No icons visible.
[/list]
[*]Detailed info with images for each icon on the sidebar. Includes clickable/selectable item links, selected-item history, and requirements to start/finish wishes, objectives, store items, etc. See [url=https://www.nexusmods.com/hollowknightsilksong/articles/141]here[/url] for more information.
[*]Multiple icon sets supported — add your own by placing “[b]Icons-*.png[/b]” files in the plugin directory.
[*][img=middle]https://www.castledragmire.com/silksong/RingSmall.png[/img] [b]Search window[/b]: Find icons by name or description. Mouse+Keyboard only.
[*][b]Saved Values Window[/b]: Watch live save data updates and link discoveries for crowdsourcing final data files.
[/list]
[*][img=middle]https://www.castledragmire.com/silksong/CompassIconSmall.png[/img] [b]Hornet Pin[/b][list]
[*]Always show Hornet’s pin — compass not required.
[*]Choose from 5 animations to highlight Hornet for better visibility.
[*]Recenter the map on your character instantly.
[/list]
[*][img=middle]https://www.castledragmire.com/silksong/SmallMarker.png[/img] [b]Markers[/b][list]
[*][img=middle]https://www.castledragmire.com/silksong/LabelIcon.png[/img]️ [b]Marker Labels[/b]: Add labels so as to never forget what a marker means.
[*]More markers: Gives 99 of every marker type.
[*]Marker mode always available, even without owning any markers.
[/list]
[*][b]Set Icon Size[/b]: Adjust icon, pin, and marker size (consistent size across zooms by default).
[/list]
[*][img=middle]https://www.castledragmire.com/silksong/MapSmall.png[/img] [b]Map Features[/b][list]
[*][b]Auto Map[/b]: Auto-reveal visited rooms once you have their map — no benches or quill required.
[*][b]Unlock Map Options[/b][list]
[*]Unlock [i]The Moss Grotto[/i] map (the beginning area) for early inventory map access.
[*]Unlock all maps.
[*]Unlock all markers.
[*][spoiler]Reveal the Abyss map.[/spoiler][/list]
[*][img=middle]https://www.castledragmire.com/silksong/HornetDashAttackSmall.png[/img] “Infinite” zooming in/out, with configurable speed, around: Markers, icons, mouse cursor, marker cursor, and the screen center.
[*]Tunable panning speeds for both normal and marker modes, with unlocked borders for full freedom.
[*]Stops map position from jumping around when switching between marker modes.
[*][i]Note:[/i] The “wide” (general-area view) map is not supported.
[/list]

[*][img=middle]https://www.castledragmire.com/silksong/DeathCacoonSmall.png[/img] Per-user-saveslot configs with automatic backups (required for marker labels).
[*][img=middle]https://www.castledragmire.com/silksong/Phonograph.png[/img] User interfaces translated [via LLM] into 21 languages.

All interfaces support 🎮️ controller, ⌨️ keyboard, and 🖱️ mouse for every feature.
[size=1](Search window does not support controller input.)[/size]
All keyboard bindings are customizable in the config.

[b]Map Mouse Controls:[/b][list]
[*]Scroll wheel — Zoom in/out
[*]Left-click drag — Pan the map
[*]Left-click icon — Show info on sidebar
[*]Hover over icon — Show title on sidebar
[*]Left-click marker or its label — Add/edit label (deleted markers retain labels until save reload). [size=1]The marker has to be in the exact correct position to restore the label.[/size]
[*]Right-click — Add/delete marker
[i]Note:[/i] Marker placement can be finicky near clustered markers — Team Cherry’s original marker auto-selection logic wasn’t built for mouse use.
[/list]
Via config you can:[list]
[*]Auto-enable mouse when sidebar or Saved Values windows are open
[*]Toggle mouse manually via the included [i]Plugin Developer Tools[/i] plugin (default: [b]F10[/b])
[/list]

[/center]Community help is crucial to nail every icon! I’ve matched 550/1,908 via my [url=https://www.nexusmods.com/hollowknightsilksong/mods/772]Pin Finder mod[/url] (error-prone) and spent countless hours on requirement/need chains, but it’s a huge dataset I can’t finish alone.

Linking values is super simple, so pitch in! With endorsements and submissions, we could map all Pharloom’s monsters, secrets, and more. Let’s do this together. 🤝🏻

[img]https://www.castledragmire.com/silksong/SavedValuesWindow.png[/img]
Use “Show Linked” on the sidebar to view icons not yet associated with save data.
[img]https://www.castledragmire.com/silksong/ValuesWindowOptions.png[/img]

To link icons and values:[list]
[*]Select an icon on the map (green highlight) and a value in the Saved Values window.
[*]Click [b]Save[/b] or [b]Save+Send[/b]:[list]
[*][b]Save[/b] → Local JSON
[*][b]Send[/b] → Uploads data to me for crowdsourcing
[/list][/list]
This will also improve icon alignment (MapGenie coordinates aren’t always precise).
If this goes well, I’ll automate for in-game JSON downloads — but there will [b]never[/b] be auto-updates or external calls without consent.

This is the only window that persists without the map open.
🎮️ Controller fully supported here too.
Some values are hard to identify — e.g., some secrets are hidden behind “*Remasker*” flags.
