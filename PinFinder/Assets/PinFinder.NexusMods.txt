[center][size=6][b]Hollow Knight Silksong: Pin Finder[/b][/size]
[size=4]Automatically scans every scene and dumps the exact position + save-flag of every collectible, bench, switch, lore tablet, etc.[/size][/center]

[center][size=5]📖️ [b]Overview[/b] 📖️[/size]
[img=top]https://www.castledragmire.com/silksong/Line425.png[/img]
[/center]Developer utility used to create accurate data files for mods like [url=https://www.nexusmods.com/hollowknightsilksong/mods/755]Pharloom Atlas[/url].
Runs once (or on demand), writes a clean JSON to the plugin folder. Process can be paused and restarted without losing anything.
You can find my latest copy of the processing, with synced changes from the last 3 months, [url=https://www.castledragmire.com/silksong/Pins.json]here[/url].
Settings dialog translated into 20 languages.


[center][size=5]⚙️ [b]Installation[/b] ⚙️[/size]
[img=top]https://www.castledragmire.com/silksong/Line425.png[/img][/center][list=1]
[*]Install [url=https://www.nexusmods.com/hollowknightsilksong/mods/26]BepInEx[/url]
[*]Extract this mod’s zip into [b]Hollow Knight Silksong/BepInEx/plugins/dakusan/[/b]
[*]You should now have (for example): [b]Hollow Knight Silksong/BepInEx/plugins/dakusan/SilkDev.dll[/b]
[*]If you use my other mods — [url=https://www.nexusmods.com/hollowknightsilksong/mods/510]Plugin Developer Tools[/url], [url=https://www.nexusmods.com/hollowknightsilksong/mods/478]NoClip[/url] — move them into the same [b]dakusan[/b] folder.
[*]Launch game → open config (F1) → click “Start Process”
[/list]
Includes my shared library: [url=https://www.nexusmods.com/hollowknightsilksong/mods/510]Plugin Developer Tools (SilkDev)[/url]
